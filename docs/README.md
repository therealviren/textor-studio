## 🚀 Overview

Textor Studio is a lightweight terminal editor focused on:

- ⚡ Speed and responsiveness  
- 🧠 Stability and reliability  
- 🎯 Distraction-free editing  
- 🔧 Custom rendering engine  
- 💾 Snapshot-based undo/redo system  

It is designed to feel simple, but still powerful enough for real development work.
